l  文件的概念、命名、组成、类型

文件的概念：文件是存储在磁盘上的一组相关信息，是文件系统中存储数据的一个命名的对象。

文件的命名：在Linux系统中，以圆点开头命名的文件是隐藏文件。

文件的组成：文件由索引节点和数据两部分组成。数据是文件的实际内容，它可以是空的，也可以非常大，并且有自己的结构。索引节点又称I节点，是文件系统中包含相应文件信息的一个记录，这些信息包括文件权限、文件主、文件大小、文件位置等。

文件的类型：普通文件、目录文件、设备文件、管道文件、链接文件。目录文件是一种特别文件，目录文件也包含数据，它是由成对的“I节点号/文件名”构成的列表。每个目录的第一项都表示目录本身，并以“点”作为它的文件名；每个目录的第二项的名字是“点点”，表示该目录的父目录。（在Linux里，文件扩展名和文件类型没有关系。）

l  常用文件操作命令

pwd

以绝对路径方式显示出当前目录

pwd

ls [选项] [目录名]

显示指定目录下的目录和文件，-a选项表示显示出所有的目录和文件，-c表示按文件的修改时间排序，-i表示显示出文件的I节点号，-l表示以长格式形式显示文件的详细信息，-h表示以易理解的格式显示文件大小，通常与-l一起使用。

若不指定目录，则显示当前目录下的内容。

ls /etc

ls -a /etc | more

ls -c /etc

ls -i /etc

ls -l /etc

ls -lh /etc

ls ..

ls .

ls

ls ./func  或者  ls func

ls -l 文件名

显示指定文件的详细信息

ls -l func/file1

ls -dl [目录名]

显示指定目录的详细信息（-d表示只显示指定目录本身，而不显示其下的各文件和目录，通常与-l一起使用。）

ls -dl /etc

ls -dl

cd a1

删除空目录

rm [-f] 文件列表

rm -r 目录列表

删除文件或目录

-f表示忽略不存在的文件，并且不给出提示信息。

-r表示递归地删除指定目录及其下属的各级子目录和文件。

rm file1 file2 file3

rm -f file1 file2 file3 file4

rm -r a1 a2

rm -r a1 a2 file1 file2 file3

rm -r *

l  文件的复制、移动、重命名

cp [-i] 源文件 目标目录

cp [-i] 源文件 目标目录/新文件名

复制文件。-i表示复制文件时若目标目录中已存在同名的文件则给出提示信息，若无-i则默认为直接覆盖。

mkdir a1 a2 a3

touch file1 file2 file3

ls

cp file1 a1

cp file1 a1/f1

ls a1

cp file1 a1

cp -i file1 a1

cp -r 源目录 目标目录

cp -r 源目录 目标目录/新目录名

复制目录。若目标目录下已存在同名的目录，则直接将目录内容合并。

cp file1 a3

cp -r a3 a1

cp -r a3 a1/a33

ls a1

cp file2 a3

cp -r a3 a1

思考1

cp file1 aaa

cp file2 aaa

cp -r a1 bbb

cp -r a2 bbb

思考2

cp file1 file2 a2

cp -r file1 file2 a1 a2 a3

总结：  cp -ri 源文件或目录 目标目录/[新文件名或目录名]

注意目标目录或文件是否已存在

mv [-i] 源文件 目标目录

mv [-i] 源文件 目标目录/新文件名

mv    源目录 目标目录

mv    源目录 目标目录/新目录名

移动或重命名文件或目录。

-i表示移动文件时若目标目录中已存在同名的文件则给出提示信息，若无-i则默认为直接覆盖。

移动目录时，若目标目录下已存在同名的目录，则移动失败。

mkdir a1 a2 a3

touch file1 file2 file3 file4

ls

mv file1 a1

mv file2 a1/f2

ls a1

mv file3 a1/file1

mv -i file4 a1/file1

 

mv a2 a1

mv a3 a1/a33

mkdir a2

mv a2 a1    //无法覆盖

思考

mkdir a1 a2 a3 a4

touch file1 file2 file3 file4

mv file1 aaa

mv file2 aaa

mv a1 bbb

mv a2 bbb

mv file3 file4 a3 a4 bbb

总结：  mv -i 源文件或目录 目标目录/[新文件名或目录名]

注意目标目录或文件是否已存在
